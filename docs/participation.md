# Participation

WTV welcomes participation from:

- wallet vendors;
- security companies;
- blockchain infrastructure providers;
- independent researchers;
- application teams that need transaction verification.

## Current Draft Questions

- Are EVM chain validation rules sufficient for wallet integration?
- Is Solana `cluster` declaration handling acceptable for security-sensitive flows?
- What minimum trust metadata should wallets publish?
- What test vectors are needed for cross-vendor interoperability?

## How to Join

- Open a GitHub discussion for broad feedback.
- Open an issue for specific spec or interoperability gaps.
- Open a PR for precise wording, tests, or reference implementation changes.

Company participation should happen through public GitHub accounts tied to real contributors.
