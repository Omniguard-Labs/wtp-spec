# WTP SDKs

中文：[WTP SDK](#wtp-sdk)

This directory groups language-specific SDK entry points.

| SDK | Path | Status |
| --- | --- | --- |
| TypeScript / JavaScript | [ts](ts) | Reference implementation wrapper over `src/` |
| Go | [go](go) | Minimal constants, metadata structs, URL helpers, and validity helpers |

The normative behavior remains defined by [../docs/specs](../docs/specs). SDKs should keep field names, Safe profile semantics, and default validity windows aligned with the specification.

## 中文

English: [WTP SDKs](#wtp-sdks)

本目录按语言整理 SDK 入口。

| SDK | 路径 | 状态 |
| --- | --- | --- |
| TypeScript / JavaScript | [ts](ts) | 基于 `src/` 参考实现的封装入口 |
| Go | [go](go) | 最小常量、metadata 结构体、URL helper 和有效期 helper |

规范行为仍以 [../docs/specs](../docs/specs) 为准。各语言 SDK 应保持字段名、Safe profile 语义和默认有效期窗口一致。
