# Code of Conduct

WTP uses the Contributor Covenant Code of Conduct, version 2.1.

## Expected Behavior

- Be respectful and precise.
- Critique ideas and text, not people.
- Disclose conflicts of interest when relevant.
- Keep security-sensitive details private until coordinated disclosure is appropriate.

## Unacceptable Behavior

- Harassment or personal attacks.
- Spam or off-topic promotion.
- Publishing private security reports without coordination.
- Misrepresenting company approval or standard status.

## Enforcement

Maintainers may moderate issues, discussions, and pull requests to keep the project productive and safe.
